# 🌐 HTML, CSS & AJAX — Web Frontend Learning Projects

> A collection of foundational web development projects covering HTML5 structure, CSS styling, responsive design with media queries, form handling, and asynchronous JavaScript patterns.

---

## 📌 About This Collection

These repositories were built during the **early weeks of my Bridgelabz apprenticeship (Jun–Jul 2021)** as part of a full-stack learning programme. Before diving into Java and automation, I built a working foundation in how the web actually works — from basic HTML structure to asynchronous data loading with AJAX.

**Repos in this collection:**
- `HTML-CSS-Intro` — First webpage: HTML structure + CSS styling
- `HTMLFORM-MediaQuery` — HTML5 forms + responsive design with media queries
- `AJAX` — Asynchronous JavaScript: callbacks, promises, and async/await
- `EmployeePayroll-App` — A complete mini web application for employee payroll

---

## 🎯 What Each Project Covers

### 🖼️ HTML-CSS-Intro — First Webpage
My very first web project. Simple, honest, and the foundation everything else builds on.

- HTML document structure: `<!DOCTYPE>`, `<head>`, `<body>`, semantic tags
- Headings, paragraphs, lists, links, images
- CSS basics: selectors, properties, colours, fonts, margins, padding
- Box model: content → padding → border → margin
- Basic page layout using `div` containers
- External CSS stylesheet linking

---

### 📋 HTMLFORM-MediaQuery — Forms & Responsive Design
A step up — building interactive forms and making pages work on different screen sizes.

**HTML5 Forms:**
- Input types: text, email, number, date, password, checkbox, radio, file
- Form validation: `required`, `min`, `max`, `pattern` attributes
- Labels, placeholders, fieldsets and legends
- Submit and reset button behaviour

**Responsive Design with Media Queries:**
- What `@media` queries are and why they matter
- Breakpoints for mobile (≤480px), tablet (≤768px), desktop (≥1024px)
- Changing layout, font size, and column structure by screen size
- Flexible images that resize with the viewport

---

### ⚡ AJAX — Asynchronous JavaScript Patterns
One of the most important concepts in modern web development — how pages load data without refreshing.

**Async Fundamentals:**
- Why synchronous code blocks the browser and why async solves this
- **Callbacks** — the original async pattern, and why callback hell is a problem
- **Promises** — `.then()` and `.catch()` chains for cleaner async code
- **`async/await`** — the modern, readable way to write async JavaScript

**Practical Usage:**
- Fetching data from a public API using `fetch()`
- Handling loading states and error states
- Updating the DOM with API response data without a page refresh
- Chaining multiple async operations in sequence

---

### 💼 EmployeePayroll-App — Full Mini Web Application
The capstone of this collection — combining HTML forms, CSS styling, JavaScript logic, and AJAX data handling into a working application.

**Features:**
- Employee registration form (name, gender, department, salary, start date, profile image)
- Form validation before submission
- Display registered employees in a styled list/table
- Edit and delete employee entries
- Local data persistence using browser storage
- Responsive layout that works on mobile and desktop

**What makes this more than just a form:**
- End-to-end user flow: add → view → edit → delete
- Input validation with user-friendly error messages
- Dynamic DOM updates without page reloads
